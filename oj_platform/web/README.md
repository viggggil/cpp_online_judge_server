# web

该目录存放平台前端静态资源。

当前页面与脚本包括：

- `home.html/js`：首页
- `problems.html/js`：题库页
- `problem.html/js`：题目详情页
- `submit.html/js`：提交页
- `submission.html/js`：提交结果页
- `submissions.html/js`：提交记录列表页
- `assignments.html/js`：作业列表页
- `assignment.html/js`：作业详情页
- `assignment-leaderboard.html/js`：作业排行榜页
- `admin-problem-create.html/js`：管理员创建题目页
- `admin-problem-edit.html/js`：管理员编辑题目页
- `admin-assignment-create.html/js`：管理员创建作业页
- `admin-assignment-edit.html/js`：管理员编辑作业页
- `auth.js`：登录/注册弹窗与登录态管理
- `nav.js`：导航与受保护跳转逻辑
- `markdown.js`：Markdown 渲染辅助
- `styles.css`：公共样式

前端采用原生 HTML + JavaScript，实现了完整的页面导航、鉴权交互、题面渲染、提交查询、作业管理与排行榜展示。
